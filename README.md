# Plant-Disease-Detection-System

A Deep Learning based web application that detects plant diseases from leaf images using TensorFlow and Streamlit.

The system predicts plant diseases in real-time by analyzing uploaded leaf images and helps in early disease identification for better crop management.

---

##  Features

- Real-time plant disease prediction
- Image upload support
- Deep Learning powered classification
- User-friendly Streamlit interface
- Confidence score visualization
- Supports multiple plant disease categories

##  Technologies Used

- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- PIL (Python Imaging Library)
- MobileNetV2
- Deep Learning
- Computer Vision

##  Dataset

Dataset used in this project was taken from the official TensorFlow dataset source.

The dataset contains thousands of plant leaf images categorized into healthy and diseased classes.

Supported plants include:
- Apple
- Tomato
- Potato
- Corn
- Grape
- Peach
- Strawberry
- Pepper
- Cherry
- Soybean
- Orange
- Raspberry
- Blueberry

and many more.

##  Deep Learning Model

This project uses:

- MobileNetV2 architecture
- TensorFlow/Keras framework
- Image preprocessing techniques
- Transfer Learning for classification

The model predicts plant diseases based on uploaded leaf images.

```bash

```


### 2️ Navigate to Project Folder

```bash

### 3️ Install Required Libraries

```bash
pip install -r requirements.txt
```

---

### 4️ Run the Streamlit Application

```bash

```
